## Frequently Asked Questions

### What's included in the free Open Source tier?

The Open Source tier includes the full InfraSpec CLI, all Gherkin step definitions, and the ability to test against real
AWS. You get everything you need to write infrastructure tests in plain English.

### How does InfraSpec Cloud billing work?

InfraSpec Cloud is billed monthly at $99/month USD. You can cancel anytime. There are no per-test charges or usage
limits - run as many tests as you want.

### Can I try InfraSpec Cloud before committing?

While we don't currently offer a free trial, we provide flexible monthly billing with no long-term commitment - cancel
anytime.

### What counts as a "team member"?

A team member is anyone who runs tests using your InfraSpec Cloud account. The Cloud plan includes 5 team members. Need
more? You can add additional team members for $20/month per seat. [Contact us](/contact) for Enterprise pricing.

### Do you offer annual billing?

Yes! Annual plans receive a 10% discount. [Contact sales](/contact) for details.

### What if I need more than 5 team members?

You can add additional team members for $20/month per seat when using InfraSpec Cloud. Our Enterprise plan includes
unlimited team members, plus additional features like SSO, self-hosting, and custom SLAs. [Contact sales](/contact) to
discuss your needs.

### How does InfraSpec stay up to date with AWS changes?

AWS releases hundreds of API changes every year. We use [CloudMirror](/docs/cloudmirror), our proprietary AWS API parity system, to automatically track changes to the AWS SDK every week. This ensures Virtual Cloud stays current with new operations and API updates. We prioritize high-impact operations (Create, Delete, Describe) that matter most for infrastructure testing. You can see exactly what's supported on our [AWS Compatibility page](/docs/compatibility).
