# Virtual Cloud Early Access

Virtual Cloud is our managed infrastructure testing service that lets you run InfraSpec tests without real AWS resources. It's currently in active development, and we're looking for early adopters to help shape the product.

## What We're Building

Virtual Cloud emulates AWS services locally, allowing you to test your Terraform configurations instantly without provisioning real infrastructure.

## Current Status

We're actively developing Virtual Cloud with support for core AWS services. Early access users will help us prioritize which services to add next and provide feedback on the developer experience.

**Currently supported services:**
- Amazon S3
- Amazon DynamoDB
- Amazon RDS
- AWS IAM
- Amazon EC2 (partial)
