# Getting Started

## Installation

The easiest way to install InfraSpec is [Homebrew](https://brew.sh).

Otherwise, you can download pre-built binaries directly from the
[GitHub releases page](https://github.com/robmorgan/infraspec/releases).

## Supported Binaries

Currently, the following binaries are provided:

- Mac (Apple Silicon)
- Mac (x64)
- Linux (x86_64)
- Linux (aarch64)
- Windows (win32-x64)
- Windows (ia32)

You can install the InfraSpec CLI via Homebrew:
