## How It Works

InfraSpec Virtual Cloud runs as a managed service that emulates AWS APIs with high fidelity. Simply point your tests
to Virtual Cloud instead of real AWS:

<Steps>
### Install InfraSpec CLI

```bash
brew install infraspec
# or
go install github.com/robmorgan/infraspec@latest
```

### Get Your API Token

Sign up at [infraspec.sh](https://infraspec.sh) and get your `INFRASPEC_CLOUD_TOKEN`.

### Run Tests with Virtual Cloud

```bash
infraspec --virtual-cloud features/
```

Virtual Cloud automatically handles all AWS API calls without touching real AWS.
</Steps>

## Usage Examples

### Basic Usage

Add the `--virtual-cloud` flag to any InfraSpec command:

```bash
# Run tests with Virtual Cloud
infraspec --virtual-cloud features/

# Run specific feature
infraspec --virtual-cloud features/s3/bucket_encryption.feature
```

### Environment Variables

Configure Virtual Cloud behavior with environment variables:

```bash
# Required: Your Virtual Cloud API token
export INFRASPEC_CLOUD_TOKEN="your-token-here"

# Optional: Custom Virtual Cloud endpoint (for enterprise)
export INFRASPEC_CLOUD_ENDPOINT="https://virtual-cloud.yourcompany.com"

# Optional: Enable debug logging
export INFRASPEC_DEBUG=true
```

## Frequently Asked Questions

### How accurate is Virtual Cloud compared to real AWS?

Virtual Cloud provides high-fidelity emulation of AWS APIs, matching real AWS behavior for 99%+ of infrastructure
testing scenarios. We continuously update our emulation based on AWS API changes and customer feedback.

### Can I test against both Virtual Cloud and real AWS?

Yes! Simply omit the `--virtual-cloud` flag to test against real AWS. Many teams use Virtual Cloud for development
and CI/CD, then validate against real AWS before production deployment.

### What happens if I exceed my plan limits?

Virtual Cloud has no usage limits‚Äîunlimited tests, unlimited parallel execution. The pricing is per team size,
not per usage.

### Do you offer volume discounts?

### Can I self-host Virtual Cloud?

### How do I migrate from LocalStack or Moto?

InfraSpec Virtual Cloud is API-compatible with standard AWS SDKs and Terraform providers. Simply point your tests
to Virtual Cloud‚Äîno code changes required. [Contact support](/support) for migration assistance.
