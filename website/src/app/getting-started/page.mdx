# Getting Started

Welcome to the Sourcegraph documentation. Here you'll find information about using and administering Sourcegraph.

<Callout type="info">
  Looking for our API documentation? Check out our [GraphQL API docs](/api).
</Callout>

## Getting starteds

<Steps>
### Install Sourcegraph

Choose how you want to deploy Sourcegraph:

- [Docker Compose](/admin/deploy/docker-compose) (recommended for most deployments)
- [Kubernetes](/admin/deploy/kubernetes)
- [Docker single-container](/admin/deploy/docker-single-container)

### Add repositories 

Connect your code hosts to sync repositories:

- [GitHub](/admin/external-service/github)
- [GitLab](/admin/external-service/gitlab) 
- [Bitbucket](/admin/external-service/bitbucket)
- [AWS CodeCommit](/admin/external-service/aws-codecommit)
- [Perforce](/admin/external-service/perforce)
- [Other Git hosts](/admin/external-service/other)

### Start searching and navigating code

- Use the search bar to find code across all your repositories
- Click on symbols to jump to definitions
- Find references and implementations
- Browse repository file trees
