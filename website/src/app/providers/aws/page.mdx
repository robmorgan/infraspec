# AWS Provider Overview

InfraSpec's AWS provider allows you to validate AWS cloud resources easily using natural language scenarios.

## Capabilities

- Validate existence and configuration of AWS resources (S3, DynamoDB, RDS, etc.)
- Assert security and compliance settings
- Test cloud infrastructure directly or as part of Terraform-managed deployments
